課題【005】三角比 (sin, cos, tan)
問1直角三角形のsin, cos, tanの求め方
1. 辺の名前を確認する 斜辺（一番長い）、底辺（角度θの隣）、対辺（角度θの向かい）を確認します。
2. 公式に当てはめる
  • sinθ = 対辺 / 斜辺 (Sの筆記体)
  • cosθ = 底辺 / 斜辺 (Cの字)
  • tanθ = 対辺 / 底辺 (tの筆記体)
※辺の長さが足りない時 三平方の定理 a² + b² = c² で計算して出してから公式を使います。
問3-4三角比の表の使い方
値を求める場合 (sin 12°など) 教科書の巻末などにある表の「角度」の列から12°を探し、横に見てsinの列の数字を答えます。
角度を求める場合 (sinθ = 0.4など) 表のsinの列の中から、0.4に一番近い数字を探し、その左にある「角度」を答えます。
問5辺の長さを式で表す
「分母」を払うイメージ たとえば cos A = AC / AB なら、両辺にABを掛けて AC = AB × cos A となります。
基本パターン
  • 底辺 = 斜辺 × cosθ
  • 高さ = 斜辺 × sinθ
  • 高さ = 底辺 × tanθ
問6-7文章題（坂道・鉄塔）
図を描く 直角三角形を描き、わかっている長さと角度を書き込みます。
どの比を使うか選ぶ
  • 斜めの距離から水平距離を出したい → cos
  • 水平距離から高さ（ビルや木）を出したい → tan
注意点：目の高さ 「目の高さ」が与えられている場合、tanで計算した高さに最後に目の高さを足すのを忘れないでください。
課題【004】不等式の応用
問1実数解をもつ・もたない条件
判別式 D を計算する D = b² - 4ac をmを使って表します。
条件に合わせて不等式を解く
  • 「実数解をもつ」 → D ≧ 0 を解く
  • 「実数解をもたない」 → D < 0 を解く
問2「すべての実数」で成り立つ不等式
グラフが浮いている条件 「常に > 0」であるためには、グラフがx軸と交わってはいけません。
計算式 D < 0 （判別式がマイナス）を解きます。
※「グラフは上にあるのに、判別式は0より小さい」という点に注意！交点がない＝解がない＝Dが負、です。
問3連立不等式
1. 別々に解く 上と下の不等式をそれぞれ解いて、xの範囲を出します。
2. 数直線を描く 2つの範囲を数直線に描き込みます。
3. 重なっている部分を答える 両方の線が重なっている範囲が答えです。
問5正方形の切り取り（文章題）
1. 変域（定義域）の確認 「辺の長さ」は必ずプラスです。
12 - 2x > 0 などから、まずxの限界（0 < x < 6など）をメモしておきます。
2. 不等式を立てる 問題文通りに「面積 ≧ 40」などの式を作り、整理して解きます。
3. 共通範囲をとる 計算して出た答えと、最初の変域の「重なっている部分」が最終的な答えです。
問6解の配置（正の部分で2点交わる）
3つの条件をすべて満たす範囲を探す 難問ですが、手順は決まっています。
  1. 判別式 D > 0 （2点で交わるため）
  2. 軸 > 0 （山頂が右側にあるため）
  3. f(0) > 0 （y切片がプラスであるため）
この3つの不等式を連立させます。
課題【003】2次不等式 計算ドリル
問1,4,8基本的な解き方・x²にマイナスがある時
最初に必ずやること x²の係数がマイナスなら、両辺に-1を掛けます。
この時、不等号の向きを逆にするのを絶対に忘れないこと！
因数分解して範囲を決める
  • (x-α)(x-β) > 0 なら、外側 (x < α, β < x)
  • (x-α)(x-β) < 0 なら、内側 (α < x < β)
問2( )² の形になる特殊なパターン
グラフがx軸に接している (x-3)² ≧ 0 などの形です。グラフをイメージします。
答えのパターン
  • > 0 (浮いてる部分) → 頂点以外のすべての実数
  • ≧ 0 (接するのもOK) → すべての実数
  • < 0 (地下にある部分) → 存在しないので「解なし」
  • ≦ 0 (0だけOK) → x = 頂点の座標（1つだけ）
課題【002】判別式とグラフ
問1重解をもつ条件
判別式 D = 0 を解く 重解＝解が1つ＝Dが0 です。
mの方程式として解き、出たmを元の式に戻してx（重解）を求めます。
重解は x = -b / 2a ですぐ求まります。
問2x軸との共有点・接するもの
y = 0 を代入して計算 ただの2次方程式として解きます。答えは座標なので (3, 0) のように書きます。
接するもの 解が1つ（重解）になったものが「接しているグラフ」です。
問3-4共有点の個数・交わる条件
判別式 D の符号を見る
  • D > 0 → 共有点2個
  • D = 0 → 共有点1個
  • D < 0 → 共有点0個
問5放物線と直線の交点
右辺同士をイコールで結ぶ x²... = 2x... のように式をつなげます。
移項して解く 全部左に寄せて2次方程式を解けば、交点のx座標が出ます。
最後に直線の式にxを代入してy座標も出しましょう。
課題【001】2次関数の決定・計算
問12次関数を求める（頂点・軸）
頂点(p, q)や軸がわかっている場合 基本形 y = a(x-p)² + q を使います。
頂点の座標を代入し、残りの1点を通る（代入）ことでaを求めます。
問33点を通る2次関数
一般形を使う 頂点が不明なので y = ax² + bx + c を使います。
連立方程式を解く 3つの点を代入して、a, b, cの連立方程式を作ります。
引き算をしてcを消去し、aとbの連立方程式に持ち込むのがコツです。
問2連立3元1次方程式
1文字を狙い撃ちで消す 式が3つありますが、係数が「1」の文字（cなど）を消しやすいです。
(1)-(2) と (2)-(3) のように2ペア計算して、同じ文字を消します。
問4-62次方程式の解き方優先順位
1. 因数分解 たすき掛けなどができるなら一番速い。
2. 解の公式 因数分解できなければ迷わず公式。
真ん中の係数が偶数なら、偶数公式（b'を使うやつ）を使うと約分不要で速いです。
